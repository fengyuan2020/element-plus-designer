# Element Plus designer 
感谢 JakHuang [form generator](https://github.com/JakHuang/form-generator) 提供的灵感。  
Element Plus设计器提供大部分element plus组件所见及所得在线设计，当然也包含表单设计。  
当前除了设置， 

## 功能概述

- element plus组件在线设计
- 组件属性更
- 导出和复制vue3代码
- 导出JSON

### 工程安装

```sh
npm install
```

### 开发

```sh
npm run dev
```

### 发布

```sh
npm run build
```
